# Alchemy Beverage Tycoon

# Overview

Alchemy Beverage Tycoon is a management and simulation game where players craft potions, manage resources, and send expedition teams to gather materials.
The game focuses on progression systems, decision-making, and resource optimization through crafting, exploration, and economy mechanics.


# Core Features
# Expedition System
Interactive map-based UI for selecting exploration locations
Players manage multiple expedition teams with different strength levels
Team strength is determined by the number of recruited units
Recruitment system with randomized outcomes based on investment tiers
Each location has:
  Different completion time
  Success rate
Players can bring potions to increase success probability (limited slots)
Time-based system (day cycle) to determine expedition completion
Outcomes:
  Success → gain crafting materials, minor losses
  Failure → heavy losses, no rewards

# Crafting & Research System
Recipe-based crafting system for potion creation
Incorrect recipes produce unknown potions with lower value
Research system tied to reputation → rank progression
Unlock new recipes using materials and currency after reaching required rank
# Inventory System
Manages:
  Materials
  Potions
  Catalysts
# Brewing Upgrade System
Upgrade brewing capabilities:
  Increase ingredient slots (from 2 → 3)
  Unlock higher temperature requirements for advanced recipes

# NPC Shop System
NPCs queue automatically to purchase potions
Each NPC has specific requests
Player interaction outcomes:
  Correct item → gain money and reputation
  Incorrect item → lose reputation
  No item → minor reputation penalty
# Dynamic Event System
Daily random events that modify potion prices
Encourages strategic selling decisions
